## Advantage 框架的优势
* 1.文件少，代码简洁
* 2.功能完善，使用简单
* 3.自带 WebView 跳转
* 4.具备较高自定义性


## Requirements 要求
* iOS 7+
* Xcode 8+


## Usage 使用方法
### 第一步 引入头文件
```
#import "OrderDic.h"
```
### 第二步 简单调用
```
[OrderDic order:dic]
```
